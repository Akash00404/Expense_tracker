# Expense_tracker
A simple and responsive Expense Tracker web application that enables users to manage their personal finances by tracking income and expenses. Users can sign up, log in, add transactions, and visualize expense summaries.

![Expense Tracker Features

- Signup and Login authentication using localStorage
- Add, view, and delete transactions (income & expenses)
- Visual summary of expenses by category (pie chart)
- Responsive, modern UI with easy navigation
- All user data is stored securely in browser localStorage

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend or database required – works entirely on the client side

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/expense-tracker.git
   cd expense-tracker
   ```

2. **Open the app:**

   Simply open the `index.html` file in your preferred web browser.

   - Or use a live server (e.g. [VSCode Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)) for best experience.

## Usage

1. **Sign Up:** Register using your email and a password.
2. **Log In:** Use the credentials to log in.
3. **Add/Remove Transactions:** Click ‘Add’ to insert income or expenses, enter details, and save.
4. **View Summary:** See visual summaries of your financial status.

## Customization

- You can add more categories, improve charts, or connect to a backend for multi-device sync.
- Styles can be modified in `style.css`.
